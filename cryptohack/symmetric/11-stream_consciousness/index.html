Challenge

In this challenge we get a program that has encrypt routine and when execute it will choose random text contant, encrypt it with CTR (without any prefix, suffix, and initial value 0) with an unknown KEY.

We need to decrypt the flag somehow

from Crypto.Cipher import AES
from Crypto.Util import Counter
import random


KEY = ?
TEXT = ['???', '???', ..., FLAG]


@chal.route('/stream_consciousness/encrypt/')
def encrypt():
    random_line = random.choice(TEXT)

    cipher = AES.new(KEY, AES.MODE_CTR, counter=Counter.new(128))
    encrypted = cipher.encrypt(random_line.encode())

    return {"ciphertext": encrypted.hex()}

Solution

Because the cipher is using the same key for all the words and also uses the same inital values for CTR (you can see this in the documentation of AES.MODE_CTR), if we know some plaintext information we can decrypt the other texts.

How do we do it? First we gather a list of all the texts in the program, we can seperate them by length (because AES-CTR is not block cipher and require no padding)

Then we can assume one of the ciphers starts with “crypto{” (that is the format of the flag), so for each text we try to get bytes that we suspect they are the key by xoring “crypto{” with the ciphertext, for each suspected bytes we try to decrypt with them the other texts, if the texts seem valid (making sense in english) we try to assume the next text.
