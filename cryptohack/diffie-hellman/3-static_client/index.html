Challenge

In this challenge we eavesdropping a session between Alice and Bob, we later can start a new session with Bob.

Solution

The vulnerabillity in this challenge is that Bob reuses his secret key b and that he allows us to choose any p and g .

At the second part of the call, we can send any g, a, p and Bob will return us B = pow(g, b, p), using that we need to find his secret b.

Let’s say we send $p = 2^{10} - 1$ and $g = 2$ then $g^{b} \bmod p$ will have one of the following values:

$$ {1, 2, 4, 8, 16, 32, 64, 128, 256, 512, 1, …} $$

$$ b \equiv_{10} 0 \rightarrow B = 1 $$$$ b \equiv_{10} 1 \rightarrow B = 2 $$$$ b \equiv_{10} 2 \rightarrow B = 4 $$$$ b \equiv_{10} 3 \rightarrow B = 8 $$$$ b \equiv_{10} 4 \rightarrow B = 16 $$$$ … $$

So for any value $x$ we can finds the reminder of $b$ with $x$! How can we use it?

I will generate all the prime numbers between 1800 to 3000.

  • Why less than 3000? because $2^{3000}$ is too much to send.
  • Why more than 1800? Because I needed that their multiplication will be close to $p$.

Then I get for each prime number $p_i$ we get $B = g^{b} \bmod 2^{p_i}$. Using that we can $log_2(B)$ to get the reminder $b \equiv_{p_i} r_i$.

Then we can use the Chinese Reminder Theorem (That was talked about in the previous article “export_grade”) to find a very large number that is a candidate for $b$. $b$ must be a multiplication of that large number to match the expected reminders!

We can just iterate over numbers and multiply them with the candidate until we find a number $k$ that matches

$$ 2^{(b_{candidate} * k)} \bmod p = B $$

Then we actualy found $b$ $$ b = b_{candidate} * k $$

Using that we can calculate the shared_secret $$ A^b \bmod p $$

and with that we can decrypt the flag!

def is_pkcs7_padded(message):
    padding = message[-message[-1]:]
    return all(padding[i] == len(padding) for i in range(0, len(padding)))


def decrypt_flag(shared_secret: int, iv: str, ciphertext: str):
    # Derive AES key from shared secret
    sha1 = hashlib.sha1()
    sha1.update(str(shared_secret).encode('ascii'))
    key = sha1.digest()[:16]
    # Decrypt flag
    ciphertext = bytes.fromhex(ciphertext)
    iv = bytes.fromhex(iv)
    cipher = AES.new(key, AES.MODE_CBC, iv)
    plaintext = cipher.decrypt(ciphertext)

    if is_pkcs7_padded(plaintext):
        return unpad(plaintext, 16).decode('ascii')
    else:
        return plaintext.decode('ascii')

def get_bob_secret(a, g, p):
    
    t = 2
    
    while True:
        try:
            r = remote('socket.cryptohack.org', 13373)

# Get primes under limit
def sieve_of_eratosthenes(limit):
    primes = [True] * (limit + 1)
    primes[0] = primes[1] = False

    for i in range(2, int(limit ** 0.5) + 1):
        if primes[i]:
            for j in range(i * i, limit + 1, i):
                primes[j] = False

    return [i for i in range(2, limit + 1) if primes[i]]
