Challenge

In this challenge we get a matrix called generator, the program creating a random value and calculates $generator^{secret}$, then it creates a random vector $v$ and calculates $w = generator^{secret} * v$

We get the generator, w, v, and we need to find the secret.

Solution

This is actualy a discrete log problem over matrices instead of numbers.

We can calculate the Jordan form of the generator matrix and get

$$ G = PJP^{-1} $$

Where $J$ is the Jordan matrix, $P$ and $P^{-1}$ are the transformation matrixes.

We need to find $k$ (the secret) such that

$$ G ^ {k} * v = w $$

So we can set the equation above and change the problem to finding an $k$ such that

$$ P * J ^ {k} * P^{-1} * v = w $$

We can multiply from left with $P^{-1}$ and get

$$ J ^ {k} * P^{-1} * v = P^{-1} * w $$

We can calculate some part of the equation and set and get $X$ and $Y$ where they both vectors $$ X = P^{-1} * v $$ and $$ Y = P^{-1} * w $$

And then we get the new equation to find $k$ for it $$ J ^ {k} * X = Y $$

After finding the Jordan form of the generator matrix we got that one of its eigenvalues appears twice at the end of the matrix

def get_jordan_blocks(J):
    blocks = []
    i = 0
    while i < J.nrows():
        j = i
        while j < J.nrows() and J[j, j] == J[i, i]:
            j += 1
            
        blocks.append(J[range(i, j), range(i, j)])
        i = j
        
    return blocks

Because we know that a Jordan matrix powered by $k$ results in each jordan_block powered by $k$ then we can look at the last jordan block after it is powered by $k$.

Given the eigenvalue at the last block (let’s call it $e$), in our last 2x2 block of the powered Jordan matrix we will get $$ J^k = \begin{bmatrix} e^k & k * e^{k-1} \ 0 & e^k \end{bmatrix} $$

Using the next equation we create above $$ J ^ {k} * X = Y $$

We can look at $J ^ k * X$

$$ J^k * X = \begin{bmatrix} e^k & k * e^{k-1} \ 0 & e^k \end{bmatrix} * \begin{bmatrix} x_0 \ x_1 \end{bmatrix} = \begin{bmatrix} x_0 * e^k + x_1 * k * e^{k-1}\ x_1 * e^k \end{bmatrix} $$

Because we know the output vector is equal to $Y$ we know that

$$ \begin{bmatrix} x_0 * e^k + x_1 * k * e^{k-1}\ x_1 * e^k \end{bmatrix} = \begin{bmatrix} y_0\ y_1 \end{bmatrix} $$

So we get 2 equations

$$ y_1 = x_1 * e^k \rightarrow y_1 * inv(x_1) = e^k $$$$ y_0 = x_0 * e^k + x_1ke^{k-1} $$

We can change the second equation to be $$ y_0 = x_0 * e^k + x_1ke^{k-1} = x_0e^k + x_1k*inv(e)*e^k $$

We can set the first equation in the second one to reduce the unknown $e^k$ elements and get $$ y_0 = x_0*(y_1 * inv(x_1)) + x_1kinv(e)*(y_1 * inv(x_1)) $$

Now we can isolate $k$

$$ y_0 - (x_0*(y_1 * inv(x_1))) = x_1kinv(e)(y_1 * inv(x_1)) $$$$ (y_0 - (x_0(y_1 * inv(x_1)))) * inv(x_1) = kinv(e)(y_1 * inv(x_1)) $$$$ (y_0 - (x_0*(y_1 * inv(x_1)))) * inv(x_1) * e = k*(y_1 * inv(x_1)) $$$$ (y_0 - (x_0*(y_1 * inv(x_1)))) * inv(x_1) * e * inv((y_1 * inv(x_1)))= k $$

So we got an equation for $k$, using this $k$ we can calculate the sha256 and find the key for the ciphertext

def decrypt_cipher(secret, ciphertext, iv):
    
    ciphertext = bytes.fromhex(ciphertext)
    iv = bytes.fromhex(iv)
    
    KEY_LENGTH = 128
    k = hashlib.sha256(str(secret).encode()).digest()[:KEY_LENGTH]
    
    cipher = AES.new(k, AES.MODE_CBC, iv)
    plaintext = cipher.decrypt(ciphertext)

    return plaintext
    
    
def get_jordan_blocks(J):
    blocks = []
    i = 0
    while i < J.nrows():
        j = i
        while j < J.nrows() and J[j, j] == J[i, i]:
            j += 1
            
        blocks.append(J[range(i, j), range(i, j)])

G = load_matrix("files\\the_matrix_reloaded\\generator.txt")

with open("files\\the_matrix_reloaded\\flag.enc") as f:
    flag_params = json.load(f)

    
with open("files\\the_matrix_reloaded\\output.txt") as f:
    output_params = json.load(f)

v = vector(GF(P), output_params["v"])
w = vector(GF(P), output_params["w"])
    
# P_transformation * J * P_transformation^-1 = G
J, P_transformation = G.jordan_form(transformation=True)
 

# J^k * P^-1 * v = P^-1 * w
# X = P^-1 * v
# Y = P^-1 * w
# J^k * X = Y
X = P_transformation.inverse() * v
Y = P_transformation.inverse() * w

jordan_blocks = get_jordan_blocks(J)
# The last eigenvalue size is 2
# jordan_blocks[-1]

e = jordan_blocks[-1][0,0]

def pinv(n):
    return pow(n, P-2, P)

x0, x1 = X[-2], X[-1]
y0, y1 = Y[-2], Y[-1]

k = (y0 - (x0 * (y1 * pinv(x1)))) * pinv(x1) * e * pinv((y1 * pinv(x1)))
k = int(k)
k = k % P

# Verify
H = G^k
print(w == H*v)

print(decrypt_cipher(k, flag_params["ciphertext"], flag_params["iv"]))
