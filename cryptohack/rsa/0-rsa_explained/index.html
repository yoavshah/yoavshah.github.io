Here I will explain RSA

RSA

RSA is an asymmetric encryption algorithm with a public key and a private key. The key generation follows

  • Choose $p$ and $q$ large prime numbers.
  • Calculate $N = p * q$.
  • Calculate the Euler’s totient function of $N$, because $N$ is multiplication of 2 prime numbers, $\varphi(N) = \varphi(p) * \varphi(q) = (p - 1) * (q - 1)$.
  • Choose exponent $e$.
  • Calculate the inverse of $e$ in $\varphi(N)$ ring (There is an inverse only if $GCD(e, \varphi(N)) = 1$), let’s call it $d$.

The public key above is $N$, $e$ The private key above is $p$, $q$, $d$

$$ c = m^e \bmod N $$$$ m = c^d \bmod N $$

RSA Proof

The proof for RSA is very easy to understand. We use the Euler-Fermat Theorem.

$$ a^{\varphi(N)} = 1 \bmod N $$

Let’s say $$ c = m^e \bmod N $$

Power both sides by $d$ $$ c^d = (m^e)^d \bmod N $$ $$ = m ^ {e*d} \bmod N $$

We know that $$ e*d = 1 \bmod \varphi(N) $$

So $$ c^d = m ^ {1 + k * \varphi(N)} \bmod N $$$$ = m * m ^ {k * \varphi(N)} \bmod N $$

Using Euler-Fermat Theorem we can conclude that $m ^ {\varphi(N)} = 1 \bmod N$ so $(m ^ {\varphi(N)}) ^ k = m ^ {k * \varphi(N)} = 1 \bmod N$

So $$ c^d = m \bmod N $$

RSA Strength

The RSA encryption strength comes from the hardness of calculating $\varphi(N)$ of a large number and from the hardness of factoring a large number to its primes.

Because the private key contains $p$ and $q$ and that $N = p*q$ we can calculate $\varphi(N)$ easly using its properties on prime numbers $\varphi(N) = \varphi(p * q) = (p - 1) * (q - 1)$
