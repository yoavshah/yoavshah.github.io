Here I will explain how to set up the enviornment for the development of smart contracts

Setting Up the Enviornment

In this tutorial we will use hardhat.

We first can create a project using the following command and choose our option

npx hardhat init

In the initialization we get an example of smart contract called Lock.sol and many other files.

In the contracts folder, there is all the code for our smart contract.

In the ignition folder, there is all the code for deploying the smart contract to the network.

In the test folder there are tests for our smart contract.

Executing a Test Enviornment

To execute a test ethereum blockchain enviornment we can just execute the following command

npx hardhat node

This command will give us multiple addresses with coins.

We can also connect our metamask extension to the local RPC server to send and receive coins.

Deploying the Smart Contract

To deploy the smart contract we can look at ignition/modules/Lock.ts

There is a javascript code that gets the contract we want to upload and then uploads it. to use this code we can just execute the following command

npx hardhat ignition deploy ./ignition/modules/Lock.ts --network localhost

The --network localhost attribute tells the code to upload it to the local ethereum network

Interacting with the Smart Contract

To interact with the smart contract we can just transfer coins using metamask, we can also use a javascript code to do such a thing.

const hre = require("hardhat");

async function main() {
  try {
    const SimpleContract = await hre.ethers.getContractFactory("Lock");

    // Connect to the deployed contract
    const contractAddress = "<ContractAddress>"; // Replace with your deployed contract address
    const contract = await SimpleContract.attach(contractAddress);

    x = await contract.<any function we want to use>();
    console.log("Updated Message:", x);

  } catch (error) {
    console.error(error);
    process.exit(1);
  }
}

main();

And to execute this script we can just use the following command

npx hardhat run ignition/modules/interact.js --network localhost
