Challenge

In this challenge we get a data encrypted with CBC mode and its randomized IV. To get the flag for this challenge we need to use the check_admin method, but use a cookie and an IV that will make the program think we has the string “admin=True;” instead of “admin=False;” (As generated in our cookie)


from Crypto.Cipher import AES
import os
from Crypto.Util.Padding import pad, unpad
from datetime import datetime, timedelta


KEY = ?
FLAG = ?


@chal.route('/flipping_cookie/check_admin/<cookie>/<iv>/')
def check_admin(cookie, iv):
    cookie = bytes.fromhex(cookie)
    iv = bytes.fromhex(iv)

    try:
        cipher = AES.new(KEY, AES.MODE_CBC, iv)
        decrypted = cipher.decrypt(cookie)
        unpadded = unpad(decrypted, 16)
    except ValueError as e:
        return {"error": str(e)}

    if b"admin=True" in unpadded.split(b";"):
        return {"flag": FLAG}
    else:
        return {"error": "Only admin can read the flag"}


@chal.route('/flipping_cookie/get_cookie/')
def get_cookie():
    expires_at = (datetime.today() + timedelta(days=1)).strftime("%s")
    cookie = f"admin=False;expiry={expires_at}".encode()

    iv = os.urandom(16)
    padded = pad(cookie, 16)
    cipher = AES.new(KEY, AES.MODE_CBC, iv)
    encrypted = cipher.encrypt(padded)
    ciphertext = iv.hex() + encrypted.hex()

    return {"cookie": ciphertext}

Solution

First, we get the encrypted cookie from the server using the get_cookie method.

We know some of the decrypted cookie characters which are “admin=False;expiry=…” and we also have control over the IV in the function check_admin.

Because of how the CBC mode is built we can use the plaintext of the cookie and the IV we recieved earlier to use the check_admin method with other IV that will change the prefix of the decrypted cookie to “admin=True;…”

So how do we do it?

After the AES block decryption routine, the output is xored with the IV and the result is the plaintext (“admin=False;…”). If we set the new IV to be [Last IV] ⊕ [Plaintext] (or at least the part we know - “admin=False;”), we get decrypted output as zeros.

Because of that, if we set the new IV to be [Last IV] ⊕ [Plaintext exists (“admin=False;…”)] ⊕ [Plaintext needed (“admin=True;”)] the decrypted output will be “admin=True;…” which is what was needed.

from Crypto.Cipher import AES
import os
from Crypto.Util.Padding import pad, unpad
from datetime import datetime, timedelta
import requests

SITE_GET_COOKIE = "https://aes.cryptohack.org/flipping_cookie/get_cookie"
SITE_CHECK_ADMIN = "https://aes.cryptohack.org/flipping_cookie/check_admin/{}/{}"

cookie = requests.get(SITE_GET_COOKIE).json()["cookie"]

iv = bytes.fromhex(cookie[:32])
enc = bytes.fromhex(cookie[32:])

needed = f"admin=True;"
exists = f"admin=False"

new_iv = b""

for i, _ in enumerate(needed):
    new_iv += (ord(needed[i]) ^ ord(exists[i]) ^ iv[i]).to_bytes(1, "little")

new_iv += iv[len(new_iv):]

print("COOKIE" + enc.hex())
print("IV: " + new_iv.hex())


flag = requests.get(SITE_CHECK_ADMIN.format(enc.hex(), new_iv.hex())).json()["flag"]

print(flag)
