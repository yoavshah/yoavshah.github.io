Challenge

In this challenge we eavesdrop a converstation between Alice and Bob that use Diffie-Hellman in the key changing but this time they choose to use the operator Add instead of Multiply! What could go wrong :) ?

Solution

Because in normal Diffie-Hellman $$ B = g^b \bmod p $$

In the Diffie-Hellman in this challenge $$ B = g*b \bmod p $$

So we could find the inverse of $g$ in $Z_p$ field using the Little Fermat Theorem. $$ g^{p} \bmod p = g \rightarrow g^{p-2} \bmod p = g_{inv} $$

After we found the inverse of $g$ we could find $b$

$$ B = g*b \bmod p \rightarrow b = B * g_{inv} \bmod p $$

After we calculated $b$ we can calculate the shared secret by multiplying $A$ with $b$

$$ secret = A * b \bmod p $$

Then we can decrypt the flag

def is_pkcs7_padded(message):
    padding = message[-message[-1]:]
    return all(padding[i] == len(padding) for i in range(0, len(padding)))


def decrypt_flag(shared_secret: int, iv: str, ciphertext: str):
    # Derive AES key from shared secret
    sha1 = hashlib.sha1()
    sha1.update(str(shared_secret).encode('ascii'))
    key = sha1.digest()[:16]
    # Decrypt flag
    ciphertext = bytes.fromhex(ciphertext)
    iv = bytes.fromhex(iv)
    cipher = AES.new(key, AES.MODE_CBC, iv)
    plaintext = cipher.decrypt(ciphertext)

    if is_pkcs7_padded(plaintext):
        return unpad(plaintext, 16).decode('ascii')
    else:
        return plaintext.decode('ascii')
    
def get_json(s):
    return json.loads(s[s.index(b"{"):])


r = remote('socket.cryptohack.org', 13380)

alice_params = get_json(r.recvline())
bob_params = get_json(r.recvline())
flag_params = get_json(r.recvline())

r.close()


# find inverse of g in p
g = int(alice_params["g"], 16)
p = int(alice_params["p"], 16)

g_inv = pow(g, p - 2, p)

B = int(bob_params["B"], 16)
b = (B * g_inv) % p

# calculate shared secret

shared_secret = (int(alice_params["A"], 16) * b) % p
dec_flag = decrypt_flag(shared_secret, flag_params["iv"], flag_params["encrypted"])
print(dec_flag)
