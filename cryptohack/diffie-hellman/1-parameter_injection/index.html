Challenge

In this challenge we are an attacker sits between Alice and Bob and we perform “Man in the Middle” attack. We need to get the flag and decrypt it.

Solution

This is an easy challenge, all we need to do is to generate our own b, then the change the message of Bob to contain our B!

Now Alice will use our B, thus we can calculate the shared_secret by calculating

pow(A, b, p)

Then decrypting the flag.

from pwn import *

import hashlib

def is_pkcs7_padded(message):
    padding = message[-message[-1]:]
    return all(padding[i] == len(padding) for i in range(0, len(padding)))


def decrypt_flag(shared_secret: int, iv: str, ciphertext: str):
    # Derive AES key from shared secret
    sha1 = hashlib.sha1()
    sha1.update(str(shared_secret).encode('ascii'))
    key = sha1.digest()[:16]
    # Decrypt flag
    ciphertext = bytes.fromhex(ciphertext)
    iv = bytes.fromhex(iv)
    cipher = AES.new(key, AES.MODE_CBC, iv)
    plaintext = cipher.decrypt(ciphertext)

    if is_pkcs7_padded(plaintext):
        return unpad(plaintext, 16).decode('ascii')
    else:
        return plaintext.decode('ascii')
